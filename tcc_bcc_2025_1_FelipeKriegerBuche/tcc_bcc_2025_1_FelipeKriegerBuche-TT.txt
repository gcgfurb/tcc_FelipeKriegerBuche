Arquivos:
/fontes/Android/Expofritz - projeto principal
/fontes/Android/jujubaSVG - biblioteca para SVG
/fontes/Unity/Expofritz - projeto principal (parte do Unity)
/fontes/Unity/unityLibrary - projeto Unity exportado em formato de biblioteca Unity

- Abrir o projeto jujubaSVG no Android Studio e buildar
- Rodar comando para publicar a biblioteca localmente:
    ./gradlew publishToMavenLocal
- Mover pasta unityLibrary para dentro da pasta do projeto Expofritz no Android
- Abrir o projeto Expofritz no Android Studio e buildar
- Executar aplicativo
